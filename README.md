# 🏏Indian-Premier-League-EDA

### 📖 Project Overview
This project focuses on analyzing Indian Premier League (IPL) match data and ball-by-ball deliveries to understand team performance, scoring patterns, and match dynamics.
The analysis follows a structured data analytics workflow — starting from raw cricket datasets, performing exploratory data analysis using Python, and identifying key insights related to team performance, scoring phases, and match outcomes.
The objective of this project is to transform raw IPL match data into meaningful insights that help understand how teams score runs, how matches progress across different phases, and which teams dominate in various match situations.

### 🎯 Purpose of the Analysis
The main goals of this analysis are to:
- Understand scoring patterns across different phases of a T20 match
- Analyze team performance and match outcomes
- Identify scoring trends across powerplay, middle overs, and death overs
- Evaluate batting performance and run distribution
- Explore how match dynamics influence final results

### 📊 Key Metrics Analyzed
- Total Runs Scored
- Boundary Count (Fours & Sixes)
- Strike Rate Trends
- Runs scored by phase (Powerplay / Middle Overs / Death Overs)
- Team-wise scoring patterns
- Match result distribution
- Over-by-over run progression

### 🗂 Dataset Information
The analysis uses IPL match datasets containing ball-by-ball and match-level information.
The datasets include:

##### 1️⃣ Matches Dataset
- Match ID
-Season
- City and Venue
- Teams participating
-Toss winner and decision
- Match winner
- Result type

##### 2️⃣ Deliveries Dataset
- Ball-by-ball match information
- Batsman and bowler details
- Runs scored on each delivery
- Extras (wide, no-ball, etc.)
- Dismissal information
- Over and ball number
  
These datasets allow detailed analysis of how runs are scored throughout a match and how teams perform across different match situations.

### 📈 Key Insights
- Run scoring significantly increases during death overs (16-20) due to aggressive batting strategies.
- The powerplay overs often show rapid scoring because fielding restrictions allow more boundary opportunities.
- Middle overs tend to stabilize scoring as teams focus on wicket preservation.
- Boundary shots contribute a significant portion of total runs in T20 cricket.
